# Storm AdBlocker

A lightweight, privacy-focused Chrome extension that blocks ads and trackers to save data and speed up browsing.

**Author:** Mr.Beauttah  
**GitHub:** [github.com/kiprutobeauttah](https://github.com/kiprutobeauttah)

## Features

- Block ads from major advertising networks
- Block tracking scripts and analytics
- Block malware and malicious domains
- Real-time statistics dashboard
- Whitelist support for trusted sites
- Minimal resource usage
- Privacy-focused (no data collection)

## Installation

### From Source

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked"
5. Select the `AdBlocker-Extension` folder
6. The extension is now installed!

### Generate Icons

1. Open `icons/generate-icons.html` in your browser
2. Right-click each canvas and save as PNG:
   - Save first canvas as `icon16.png`
   - Save second canvas as `icon48.png`
   - Save third canvas as `icon128.png`
3. Place all PNG files in the `icons/` folder

## Usage

1. Click the extension icon in your browser toolbar
2. View real-time statistics of blocked content
3. Toggle blocking on/off with the switch
4. Whitelist current site if needed
5. Reset statistics anytime

## Statistics Tracked

- Ads Blocked
- Trackers Blocked
- Data Saved (estimated)
- Sites Visited

## Privacy

This extension does NOT:
- Collect any personal data
- Track your browsing history
- Send data to external servers
- Require account registration

All data is stored locally on your device.
